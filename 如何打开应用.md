# 如何打开「需求 → 测试用例流水线」应用

## 方式一：终端一条命令（推荐）

在终端中执行：

```bash
cd /Users/wangqilong/Desktop/自动化工作
pip install streamlit PyYAML -q
python3 -m streamlit run app_ui.py --server.port 8501
```

终端里会显示类似：

```
  You can now view your Streamlit app in your browser.

  Local URL: http://localhost:8501
  Network URL: http://192.168.x.x:8501
```

**在浏览器中打开：http://localhost:8501** 即可使用。

---

## 方式二：双击/运行脚本

- **Mac / Linux**：在终端执行  
  `./run_app.sh`  
  或在 Finder 中右键 `run_app.sh` → 用终端打开。

- **Windows**：双击 `run_app.bat`。

然后浏览器访问：**http://localhost:8501**

---

## 首次使用前

若未安装依赖，请先执行：

```bash
cd /Users/wangqilong/Desktop/自动化工作
pip install -r requirements.txt
```

---

## 使用步骤

1. 打开 **http://localhost:8501**
2. 在「运行流水线」页填写 **Quip 文档链接**、**Quip Token**、**Gemini API Key**
3. 点击「运行流水线」，等待执行完成
4. 查看 Agent 沟通过程、最终结果，以及 Excel 下载 / Quip / Google 表格链接

关闭应用：在运行 `streamlit run` 的终端里按 `Ctrl+C`。

若启动时提示 **Port 8501 is not available**，说明本机已有实例在跑，直接在浏览器打开 **http://localhost:8501** 即可。
